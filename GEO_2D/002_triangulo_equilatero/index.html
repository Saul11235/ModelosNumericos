Triangulo equilatero

Descripción: esta librería abstrae el modelo de un triángulo equilatero en el campo euclídeo de dimensiones reales (no complejas).

Autor: Edwin Saul

Tamaño: 1.1Kb

Descargar

Variables:

Variable Nombre Variable Unidad Descripción Rango esperado
A Area unidad de superficie Área del cuadrado mayor o igual a 0
L Lado unidad de longitud Longitud del lado del cuadrado mayor o igual a 0
P Perímetro unidad de longitud Longitud del Perímetro del cuadrado mayor o igual a 0
D Diámetro unidad de longitud Longitud del Diámetro del cuadrado mayor o igual a 0

Ecuaciones del modelo:

A=L^2
P=4*L
D=L+3^0.5

Anotaciones:

El presente modelo se basa sólo en longitudes y distancias positivas, cualquier otro valor no dará resultados satisfactorios.
